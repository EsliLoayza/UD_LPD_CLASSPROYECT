Semana 04: ¡Tu Primera Base de Datos en Memoria con `DataTable`!

¡Un Salto Gigante!

Hoy dejamos atrás las listas simples para abrazar el poder de los **datos estructurados**. Aprenderás a construir una tabla de datos completa, con columnas tipadas y filas, directamente en la memoria de tu programa. ¡Es la antesala al mundo de las bases de datos reales!

Conceptos Clave Desglosados

¿Qué es un DataTable?

Es tu **hoja de cálculo de Excel programable**. Un objeto que representa una tabla con columnas (`DataColumn`) y filas (`DataRow`). Es el corazón de la gestión de datos desconectada en ADO.NET.

Definiendo la Estructura: DataColumn

Cada columna se define con un nombre y, lo más importante, un tipo de dato. Esto garantiza la integridad de tu información.

// Definimos una columna para texto y otra para números
miTabla.Columns.Add("Producto", typeof(string));
miTabla.Columns.Add("Stock", typeof(int));
Agregando Datos: DataRow

Las filas se crean a partir de la estructura de la tabla y luego se rellenan y agregan a la colección de filas de la tabla.

// 1. Crear una fila con la estructura correcta
DataRow nuevaFila = miTabla.NewRow();

// 2. Rellenar los datos
nuevaFila["Producto"] = "Laptop";
nuevaFila["Stock"] = 50;

// 3. Agregar la fila a la tabla
miTabla.Rows.Add(nuevaFila);
La Conexión Mágica: DataGridView

Este control visualiza tu `DataTable` automáticamente. ¡Cualquier cambio en el `DataTable` se refleja en la grilla al instante!

// ¡Esta sola línea lo hace todo!
miGrilla.DataSource = miTabla;

Actividad de Aprendizaje N° 04: ¡A Construir!

Ejercicio: Mini Gestor de Alumnos

Crearás una aplicación para gestionar alumnos. Deberá tener campos de texto para DNI, Nombres, Apellidos y Edad. Con un botón "Agregar", los datos se guardarán en un `DataTable` y se mostrarán en un `DataGridView`. Un segundo botón "Eliminar" quitará de la lista al alumno que esté seleccionado en la grilla.
