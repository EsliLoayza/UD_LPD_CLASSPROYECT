Diagrama de Clases Script SQL

Cada clase se convierte en una tabla. Los atributos son las columnas. La relación 1-a-Muchos se crea con una Clave Foránea.

-- De la Clase 'Articulo'
CREATE TABLE articulos (
  id_articulo INT PRIMARY KEY AUTO_INCREMENT,
  titulo VARCHAR(255) NOT NULL,
  contenido TEXT,
  fecha DATETIME
);

-- De la Clase 'Comentario'
CREATE TABLE comentarios (
  id_comentario INT PRIMARY KEY AUTO_INCREMENT,
  texto VARCHAR(500),
  fecha DATETIME,
  id_articulo INT,  -- Clave Foránea
  FOREIGN KEY (id_articulo) REFERENCES articulos(id_articulo)
);
