Semana 05: Construyendo el Puente de Datos

De la memoria volátil al almacenamiento persistente con ADO.NET y MySQL.

Arquitectura Cliente/Servidor

Nuestra aplicación (Cliente) le pedirá datos a un programa especializado (Servidor de Base de Datos). ¡Así funcionan casi todas las aplicaciones del mundo!

CLIENTE

(Tu App en C#)

Petición (SQL)

Respuesta (Datos)

SERVIDOR

(Base de Datos MySQL)

El Proceso en 4 Pasos

1

Instalar el Conector

Añadimos el "traductor" de MySQL a nuestro proyecto usando el Administrador de Paquetes NuGet.

2

Crear la Conexión

Establecemos la "tubería" hacia la base de datos usando la Cadena de Conexión (la dirección del servidor).

3

Ejecutar la Consulta

Enviamos al "mensajero" (`DataAdapter`) con una pregunta en SQL para que traiga los datos.

4

Llenar el DataSet

Los datos regresan y se guardan en nuestro "almacén" local (`DataSet`) para luego mostrarlos.

Constructor de Cadena de Conexión

Tu código C#:

string cadena = "server=127.0.0.1;database=mi_basedatos;uid=root;pwd=admin;";
