Semana 03: ¡Manejando Listas! Controles Avanzados y Colecciones.

¡Hora de Organizar la Información!

Las aplicaciones del mundo real manejan listas de cosas: clientes, productos, tareas. Hoy aprenderás a usar los controles que te permitirán mostrar y manipular estas listas, dando a tus programas una capacidad de gestión de datos mucho más potente y profesional.

1. Marco Teórico: El Mundo de las Colecciones

a. ListBox y ComboBox: Tus Nuevas Herramientas

Son controles diseñados para manejar colecciones de datos.

  • ListBox: Muestra una lista de ítems en una caja visible. Ideal para mostrar varias opciones a la vez.
  • ComboBox: Muestra una lista desplegable, ahorrando espacio en pantalla.

Ambos usan la colección Items para manipular sus datos. Los métodos principales son:

  • .Items.Add("Nuevo") - Agrega un ítem.
  • .Items.Remove("Existente") - Elimina un ítem.
  • .Items.Clear() - Borra todos los ítems.

b. MenuStrip: Navegación Profesional

Este control te permite crear la barra de menú superior ("Archivo", "Editar", etc.) que toda aplicación de escritorio profesional debe tener. Organiza las acciones de tu programa de forma lógica y familiar para el usuario.

c. El Bucle `foreach`: Recorriendo Colecciones

Es la forma más sencilla y segura de "visitar" cada elemento dentro de una colección, uno por uno, para poder procesarlos. 

// 'item' tomará el valor de cada elemento de la colección en cada pasada del bucle
foreach (string item in miListBox.Items)
{
    // Procesar el 'item' actual
    Console.WriteLine(item);
}

2. Actividad de Aprendizaje N° 03

Ejercicio: Gestor de Tareas Pendientes

Construye una aplicación de "To-Do List". Deberá tener una sección para ingresar nuevas tareas, una lista para tareas "Pendientes" y otra para "Completadas". El usuario debe poder agregar tareas y moverlas entre ambas listas. Incluye un menú para limpiar todas las tareas y para salir de la aplicación.
